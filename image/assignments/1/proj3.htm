In this assignment you will work with XML/XSLT and Google chart tools to create a graphical markup language (GML) which allows users to create graphs such the one shown below:

 

 

 

Google offers extensive support for adding various types of graphical tools (mostly based on jQuery) to web sites using JavaScript API’s which are not easy to use for the average web developer. For example, the graph shown above is created using an html source file which assumes moderate familiarity with JavaScript.

 

Your task in this assignment is to create a markup language which allows users to specify the type of graph they want to create in their web page using your graphical XML called GML. For example, the following GML code is the exact representation of what is specified programmatically in the source file.

 

<?xml version="1.0"?>

 

<charts num_charts="1">

 

   <chart type="pie" width="400" height="300" align="left" id=”chart1_div”>

       <title>How Much Pizza I Ate Last Night</title>

   

       <columns>

            <column name="Topping" type="string"/>

            <column name="Slices"  type="number"/>      

       </columns>

   

       <rows>

            <row name="Mushrooms" value="3" />

            <row name="Onions" value="1" />  

            <row name="Olives" value="1" />       

            <row name="Zucchini" value="1" /> 

            <row name="Pepperoni" value="2" />         

       </rows>

   </chart>

  

</charts>

4.    Write an XSLT program that parses your GML code using the runXML utility (browsers do not work here) and produces the appropriate HTML output.

 

5.    Note that every chart is drawn in its own division with a specific ID (e.g. chart1_div, chart1_div, etc.)
