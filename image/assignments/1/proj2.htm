In this assignment you will develop SAX (SAXProg.java) and DOM (DOMProg.java) tools to create custom-designed HTML photo albums as described below:

 

·         Assume your programs have access to a directory called images which contains a number of images

 

·         Your programs will SAX or DOM parse a source XML document to create the actual HTML photo album pages

 

·         For example, given the XML source document called myalbum.xml shown below:

 

<?xml version=”1.0”?>

 

<!-- navigational links to all the album pages must appear on bottom of the pages -->

<!-- every image has the same width and height -->

<album directory=”images” links=”bottom” width=”200” height=”100”>

 

  <!-- there is going to be  two pages in your album -->

  <pages number=”2”>

 

       <!—the first page is to be called page1.html and has a 2 by 1 grid

       <page name=”page1.html” rows=”2” cols=”1”

                   background_color=”orange” title=”First day of my camping trip!”>

 

               <img src=”pic1.jpg”>

                    <!—description of  the image -->

                    Welcome to the Wonderful World of Camping!

               </img>

 

               <img src=”picxz.jpg”>

                     The best place to camp in California

               </img>

       </page>

 

       <!—the first page is to be called page2.html and has a 2 by 2 grid

       <page name=”page2.html” rows=”2” cols=”2”

                   background_color=”orange” title=”Last day!”>

 

               <img src=”pic7.jpg”>

                    My RV

               </img>

 

               <img src=”picgh.jpg”>

                     Nature at its best

               </img>

 

               <img src=”pic5.jpg”>

                    Beautiful sunset

               </img>

 

               <img src=”picasq.jpg”>

                     Campfire was fun!!!

               </img>

      </page>

 

 

  </pages>

</album>

 

   

You would use either DOMProg.java or SAXProg.java to create the photo album pages as follows:

 

% javac DOMProg.java

% java  DOMProg myalbum.xml             -- this would create the files page1.html and page2.html

 

Or, alternatively:

 

% javac SAXProg.java

% java  SAXProg myalbum.xml              -- same outputs as above

 

 

The pages can then be opened using your favorite browser and should look like this:
